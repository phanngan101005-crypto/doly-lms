Lưu ý Kỹ thuật

MụcChi tiết
EngineInnoDB — hỗ trợ khoá ngoại, transactions, row-level locking
Charsetutf8mb4 + utf8mb4_unicode_ci — hỗ trợ tiếng Việt và emoji đầy đủ
IndexĐã tạo index cho tất cả cột thường xuyên truy vấn (WHERE, JOIN, ORDER BY)
FK PolicyON DELETE CASCADE cho bảng con · ON DELETE SET NULL cho bảng tuỳ chọn
Bảo mậtpassword_hash → bcrypt (VARCHAR 255) · auth_token → ngẫu nhiên 255 ký tự
ENUMSử dụng ENUM cho các trường có giá trị cố định (role, status, type)
Timestampscreated_at: DEFAULT CURRENT_TIMESTAMP · updated_at: ON UPDATE CURRENT_TIMESTAMP
DOLY LMS — Database Schema v1.0 · Thiết kế bởi Qoder AI · MySQL 8.x
